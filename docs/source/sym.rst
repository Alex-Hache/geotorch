Symmetric Matrices: :math:`\operatorname{Sym}(n)`
========================================================

.. currentmodule:: geotorch.sym

:math:`\operatorname{Sym}(n)` is the vector space of symmetric matrices:

.. math::

    \operatorname{Sym}(n) = \{X \in \mathbb{R}^{n \times n}\:\mid\: X^\intercal = X \}
